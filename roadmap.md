# Images on Breads

We need that users can upload an image when creating / editing a bread and store an image related to the bread. This image should be shown in the following views
- breads
- customer 
- customer_daily_defaults
- bread (option to also change the image)
- index

Think very carefully on how to show the images in the UI, because the number of breads will be between 5 and 15. Write a detail plan for this in a feature.md file.
